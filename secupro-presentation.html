02 — Fonctionnalités

Ce que SecuPRO fait concrètement

8 modules opérationnels, tous connectés à la même base de données Supabase, accessibles via une interface web responsive.

Module 01
Gestion des agents
Création, modification, désactivation. Suivi carte pro CNAPS, qualifications (CQP APS, SSIAP, SST), statut actif/inactif/suspendu.
Module 02
Import CSV en masse
Wizard 5 étapes. Validation Zod stricte. Import de 50+ agents en 30 secondes. Logs d'erreur détaillés dans import_audits.
Module 03
Dashboard performance
KPIs en temps réel : heures réalisées vs contrat, taux d'absentéisme, note client, infractions. Graphiques mensuels recharts.
Module 04
SecuIA — Assistant IA
Assistant Claude Sonnet spécialisé IDCC 1351 et réglementation CNAPS. Réponses en 3 secondes sur les droits et obligations.
Module 05
Gestion des contrats
CDI, CDD, temps partiel. Coefficient grille IDCC 1351, taux horaire, dates. Alertes fin de CDD automatiques.
Module 06
Stripe & Abonnements
Checkout sécurisé, webhook temps réel, essai 7 jours. Gating des features par plan. Mise à jour instantanée après paiement.
